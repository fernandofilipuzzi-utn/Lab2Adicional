https://learn.microsoft.com/en-us/sql/reporting-services/report-server/configure-web-portal?view=sql-server-ver16
Open the RSReportServer.config file in a text editor. By default, the file is located at 
        \Program Files\Microsoft SQL Server\MSRS12.\<instancename>\Reporting Services\ReportServer.
Find ReportServerURL.
Replace it with the URL of the report server instance.
Save your changes and close the file.
